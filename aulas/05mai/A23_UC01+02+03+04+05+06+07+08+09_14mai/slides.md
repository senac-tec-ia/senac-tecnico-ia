# Av.03 - TokenLab (Recuperacao)

**Quem precisa:** PA ou NA em UC05 Ind.1 ou UC03 Ind.1+2

**O que fazer (Fases 1 e 2 apenas):**
- `input()` para capturar texto do usuario
- Calcular tokens: `len(texto) // 4` (minimo 1)
- Classificar tamanho: curto / medio / longo
- Calcular custo de 1 chamada em 3 modelos:
  - GPT-4: `tokens * 0.30 / 1000`
  - GPT-3.5: `tokens * 0.03 / 1000`
  - Gemini: `tokens * 0.01 / 1000`
- Imprimir os 3 custos

**Nao precisa:** Fases 3 e 4 (volumes, recomendacao, context window)

**Entrega:** link do Colab ou GitHub · ate 23h59
