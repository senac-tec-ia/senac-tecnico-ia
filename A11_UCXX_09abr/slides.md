# IA que Resolve: O Que É uma Solução de IA?

| Empresa | Problema | Como a IA resolve |
|---|---|---|
| **Nubank** | Fraude no cartão | IA detecta padrão suspeito em tempo real e bloqueia automaticamente |
| **iFood** | Entregador certo para o pedido | IA calcula rota, demanda e proximidade para alocar o entregador ideal |
| **Embrapa** | Identificar praga na lavoura | IA analisa foto da folha e classifica a doença com alta precisão |

> **Em todos os casos:** um **problema real** + **dados disponíveis** + uma **IA treinada para aprender com esses dados**.

# Padrões que Emergiram

<!-- SLIDE 11 — Anatomia de uma Solução IA -->

# Anatomia de uma Solução IA

*O que os cases de hoje revelaram:*

| Etapa | O que acontece |
|---|---|
| **1.** Problema | Identifica um desafio real que tem dados associados |
| **2.** Dados | Coleta e organiza exemplos (histórico, fotos, textos) |
| **3.** Modelo | IA aprende padrões com esses dados (treino) |
| **4.** Resultado | IA faz previsões ou decisões em casos novos |
| **5.** Impacto | Melhoria real: velocidade, custo, precisão ou experiência |

> Toda IA que você vai encontrar na carreira segue esse caminho — com mais ou menos complexidade.

<!-- SLIDE 17 — Estilos de Título: Por Que Usar H1 e H2? -->

# Estilos de Título: Por Que Usar H1 e H2?

**Não é só estético — é estrutural.**

- **Título 1 (H1)** → seção principal (ex: "Glossário de IA", "Minha Opinião")
- **Título 2 (H2)** → subseção, se houver

**Por que usar estilos e não só negrito?**
1. Word gera **sumário automático** a partir dos estilos
2. Mantém consistência visual em todo o documento
3. É como profissionais formatam relatórios reais

**Como aplicar:** Página Inicial → painel de Estilos → clique em **Título 1**

# Como Ler um Termo Técnico Sem Tradutor

**3 passos para entender qualquer termo de IA:**

1. <carbon-search /> **Contexto** — Em que frase aparece? O contexto revela o significado.

2. <carbon-growth /> **Raiz da palavra** — Muitos termos têm raiz latina que você conhece.

3. <carbon-repeat /> **Tradução funcional** — Não é literal: _"O que isso faz na prática?"_

> **Atalho:** pesquise `[termo] in machine learning` no Google.
