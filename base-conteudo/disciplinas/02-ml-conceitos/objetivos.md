# Objetivos de Aprendizagem — Disciplina 2: Fundamentos de Machine Learning

## Objetivos Gerais

Ao final desta disciplina, o aluno será capaz de:

1. Diferenciar Machine Learning de programação tradicional
2. Classificar um problema de ML pelo tipo de aprendizado (supervisionado, não supervisionado, por reforço)
3. Descrever as etapas do pipeline de ML de ponta a ponta
4. Explicar o conceito de overfitting e a importância da divisão treino/teste
5. Interpretar métricas básicas de avaliação: acurácia, precision, recall e F1

## Competências Desenvolvidas

- Pensamento orientado a dados
- Formulação de problemas reais como problemas de ML
- Vocabulário técnico de ML em PT-BR
- Base conceitual para as aulas práticas de Python e Scikit-learn

## Avaliação

- Quiz de fixação (5 perguntas de múltipla escolha) ao final da aula
- Exercício: dado um problema real, identificar tipo de ML e métricas adequadas
