## Diretrizes metodológicas

- **Articulação Matemática + Física + Sociologia:** cada bloco conecta explicitamente com uma dessas áreas
- **LGPD presente:** sempre que dados pessoais aparecerem, conectar com LGPD
- **Progressão Windows→Word→Excel:** não saltar etapas — Ind. 1→2→3

- **Nomenclatura `snake_case`:** reforçar sempre que criar arquivos de exemplo nos slides
