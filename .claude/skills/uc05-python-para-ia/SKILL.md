# Skill: UC05 — Desenvolvimento de Linguagem Python

**Código:** UC05 | **Carga Total:** 67h (80 HA) | **Peso no dia:** Pesado (3 HA/bloco, Bloco 1)

Contexto vivo em: `.github/agents/contextos/contexto-python-para-ia.md`

---

## Diretrizes metodológicas

- **Cotidiano primeiro, IA depois:** introduzir todo conceito novo com analogia adolescente (playlist, chamada, jogo) ANTES do contexto IA
- **Nível 0 em cada bloco:** toda sequência começa com exercício de LEITURA ("leia e preveja a saída") antes de blank ou escrita do zero
- **Código starter obrigatório:** todo exercício entrega a assinatura da função pronta — alunos nunca começam de arquivo em branco
- **Pair programming:** exercícios de código funcionam em pares — adaptar dinâmicas para revisão em dupla
- **Ambientes práticos:** referenciar onde rodar (Google Colab ou VS Code) em toda aula
- **Erros fazem parte:** incluir slides de leitura de `SyntaxError`, `TypeError`, `NameError` — Ind. 3 começa no T1

## Sequência de exercícios por conceito

1. **Nível 0** — leitura: dado o código, preveja a saída
2. **Nível 1** — complete o blank: código quase pronto com `___` para preencher
3. **Nível 2** — escreva a função a partir da assinatura dada
4. **Nível 3** — mini-projeto contextualizado em IA
